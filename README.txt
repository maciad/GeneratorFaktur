1. Sposób uruchomienia programu
Program uruchamia się z linii poleceń, w katalogu, w którym znajduje się plik wykonywalny, wpisując polecenie:
    java -jar GeneratorFaktur-1.0-SNAPSHOT-jar-with-dependencies.jar [plik-wejściowy.csv/xlsx] [plik-wyjściowy.xml]
Ważne jest, aby plik wejściowy znajdował się w tym samym katalogo lub została podana pełna ścieżka do niego.
